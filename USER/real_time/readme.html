AVRgcc - KS0108 Library - ReadMe

Version 1.1

Die Library bietet ein paar elementare Grafikfunktionen und unterstützt verschiedene Schriftarten. Die Schriftarten können mit meinem FontEditor erstellt werden.

Die Dokumentation lässt noch etwas zu wünschen übrig. Wenn ich mal etwas Zeit erübrigen kann wird sie aber noch vertieft :) Das beiliegende Demoprogramm zeigt den Umgang mit den Fonts, die restlichen Funktionen sollten dann eigentlich einigermaßen selbsterklärend sein. Ansonsten stehe ich natürlich für Fragen zur Verfügung. Weitere Informationen und der FontEditor sind auf meiner Homepage zu finden.

Das Demo-Programm ist für den mega128 geschrieben. Um es für einen anderen AVR zu kompilieren müssen aber nur der MCU-Typ im makefile und die Port-Definitionen in der ks0108.h angepast werden.

Lizenz: gibts nich wirklich. Wer meine Lib in einem privaten Projekt verwenden oder verändern möchte soll das tun. Bei einem kommerziellen Projekt würde ich mich über eine Rückfrage freuen. In jedem Fall muss aber mein Copyright erhalten bleiben.
